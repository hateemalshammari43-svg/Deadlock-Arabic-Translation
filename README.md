## ✨ مميزات التعريب / Features

### باللغة العربية:
* **تعريب شامل:** يغطي القوائم الرئيسية، أسماء وقدرات الأبطال، تفاصيل العناصر (Mods)، والخصائص.
* **حماية الأكواد:** الحفاظ على متغيرات اللعبة (مثل `{s:AbilityCooldown}`) وتنسيقات الألوان حتى لا تسبب أي مشاكل برمجية أو انهيار للعبة.
* **تحديثات مستمرة:** مواكبة تحديثات اللعبة الدورية وإضافة النصوص الجديدة.

### In English:
* **Comprehensive Translation:** Covers main menus, hero names & abilities, item (Mods) details, and attributes.
* **Safe formatting:** Preserves in-game variables (e.g. `{s:AbilityCooldown}`) and HTML tags to prevent UI breakage or game crashes.
* **Continuous Updates:** Staying up-to-date with Valve's game patches and localizing newly added keys.

## 🚀 طريقة التركيب / How to Install

> [!IMPORTANT]
> التعريب يعتمد على استبدال ملفات اللغة الإنجليزية الافتراضية، لذا يجب أن تكون لغة اللعبة في Steam مضبوطة على **الإنجليزية** (English).
> 
> The translation replaces the default English files, so make sure the game language on Steam is set to **English**.

### خطوات التثبيت / Installation Steps:
1. قم بتحميل أحدث ملف تعريب مضغوط من سيرفر الديسكورد أو من قسم الـ [Releases](#) (عند توفره).
2. قم بفك الضغط عن الملف، وستجد مجلداً باسم `game`.
3. انسخ مجلد `game` وتوجه إلى مسار اللعبة الرئيسي على جهازك، وهو غالباً:
   ```text
   C:\Program Files (x86)\Steam\steamapps\common\Deadlock
   ```
4. قم بلصق المجلد هناك ووافق على استبدال جميع الملفات عند ظهور نافذة التأكيد.
5. افتح اللعبة واستمتع بالتعريب!
